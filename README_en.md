# Simple Slip Captcha

A simple sliding captcha based on Java Spring Boot + Vue3.

## Project Overview

Simple Slip Captcha is a project that provides sliding captcha functionality, built with the Spring Boot and Vue3 frameworks. It's easy to use and can generate basic sliding captchas for simple identity verification flows in websites or applications.

⚠️ **Note: This project does not include behavioral model detection—it only performs basic coordinate offset validation and trajectory verification. Do not use it in scenarios requiring high security.**

## Tech Stack

- Java 8+
- Spring Boot
- Maven
- Vue3

## Features

- Core components implemented using native DOM, making them easy to integrate into other front-end frameworks.
- Supports touch events for compatibility with mobile devices.
- Configurable parameters including image resolution, text content, noise texture density, color schemes, and more.
